Cumulative Distribution Function (CDF)

Accumulating probabilities across the range of a random variable.

Definition

CDF

The Cumulative Distribution Function \(F_X(x)\) of a random variable \(X\) is defined as the probability that \(X\) will take a value less than or equal to \(x\): $$ F_X(x) = P(X \leq x) $$

Properties

  • Non-decreasing: If \(a < b\), then \(F_X(a) \leq F_X(b)\).
  • Limits: $$ \lim_{x \to -\infty} F_X(x) = 0 \quad \text{and} \quad \lim_{x \to \infty} F_X(x) = 1 $$
  • Right-Continuous: \( \lim_{\epsilon \to 0^+} F_X(x + \epsilon) = F_X(x) \).
  • Interval Probability: $$ P(a < X \leq b)=F_X(b) - F_X(a) $$

Discrete vs. Continuous

Discrete Random Variables

The CDF is a step function. It jumps at each value \(x\) where \(P(X=x) > 0\). The size of the jump is equal to the probability \(P(X=x)\).

Continuous Random Variables

The CDF is a continuous function. It is the integral of the Probability Density Function (PDF): $$ F_X(x) = \int_{-\infty}^x f_X(t) \, dt $$
