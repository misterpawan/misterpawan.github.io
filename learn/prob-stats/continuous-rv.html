Continuous Random Variables

Variables that can take any value within a range.

Definition

A random variable \(X\) is continuous if its CDF \(F_X(x)\) is a continuous function. Unlike discrete variables, the probability of taking any single exact value is 0: \(P(X=x) = 0\).

Probability Density Function (PDF)

Definition

The PDF \(f_X(x)\) is the derivative of the CDF: $$ f_X(x) = \frac{d}{dx} F_X(x) $$

Properties

  • \( f_X(x) \geq 0 \) for all \(x\).
  • \( \int_{-\infty}^{\infty} f_X(x) \, dx = 1 \).
  • \( P(a < X \leq b)=\int_a^b f_X(x) \, dx \).

Expectation and Variance

Expected Value

$$ E[X] = \int_{-\infty}^{\infty} x f_X(x) \, dx $$

Variance

$$ \text{Var}(X) = \int_{-\infty}^{\infty} (x - E[X])^2 f_X(x) \, dx $$

Method of Transformation

If \(Y = g(X)\) where \(g\) is a strictly monotonic and differentiable function, then the PDF of \(Y\) is:

$$ f_Y(y) = f_X(x) \left| \frac{dx}{dy} \right| $$ where \(x = g^{-1}(y)\).
